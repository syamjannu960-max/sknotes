# **App Name**: CourseNote

## Core Features:

- Course Listing: Display a list of all available courses for users to browse.
- Hierarchical Navigation: Allow users to navigate through courses, semesters, chapters, and units.
- Content Display: Display chapter titles, rich text content, and PDF download button.
- Admin Authentication: Secure admin login with email and password.
- Content Management: Enable admin to add, edit, and delete courses, semesters, chapters, and units.
- PDF Upload: Allow admins to upload PDF files for chapters, stored securely.
- AI Content Enrichment: Allow admins to enrich chapter content with suggested topics generated by an AI tool based on the existing text content.

## Style Guidelines:

- Primary color: Moderate blue (#6699CC) for a learning and reliable feel.
- Background color: Light blue (#E6F0FF), same hue as primary color but desaturated, in order to suggest the same vibe in a subtle way.
- Accent color: Violet (#9966CC), an analogous color for highlighting and calls to action.
- Headline font: 'Space Grotesk' (sans-serif) for titles; body font: 'Inter' (sans-serif) for content. 'Space Grotesk' provides a techy feel that pairs well with the modern style of 'Inter', which is suitable for reading at length.
- Clean and responsive layout for optimal viewing on different devices.
- Simple, consistent icons for navigation and actions.
- Subtle transitions and animations for a smooth user experience.
